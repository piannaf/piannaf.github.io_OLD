University Course Information System

Jul 13, 2010

A university course information system implemented in JSP with MySQL connectivity. We had a choice of JSP or PHP and I chose JSP because of the two, I prefer Java.

The focus of this project was to implement a course scheduling algorithm to minimize clashes. I took a naive approach to the algorithm and a very inefficient way of dealing with database connections. It is a proof of concept that I am able to design and program a complex information system requiring database connectivity, authentication and automation.
