Basic Unix Commands from Scratch

Nov 26, 2010

As a warm-up exercise for this class on Operating System Architecture, we had to implement ‘ls’, ‘cp’, and ‘rm -rf’ as built-in commands to a custom shell using direct system calls instead of the C-library.

The ‘ls’ command needed to print its result in a specific way and the ‘cp’ command wasn’t supposed to deal with multiple files or directories.
