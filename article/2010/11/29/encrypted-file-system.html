Encrypted File System

Nov 29, 2010

Talk about a deceptively simple assignment. Two parts: implement ‘immediate files’ and ‘encryption’.

There were times when a single line of code took hours to research. The lecture content was basically: there exists a VFS and data is stored in blocks. I walked the Linux source tree down from the open and write system calls through the VFS layer to the block device driver to understand this stuff and it was a lot of fun.
